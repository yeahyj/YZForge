# 07. API 示例

## Module

```ts
import { Module } from 'yzforge';
import { HomeFlow } from './flow/HomeFlow';
import { HomeService } from './service/HomeService';

export class HomeModule extends Module {
    protected async onLoad(): Promise<void> {
        this.useService(HomeService);
    }

    protected async onEnter(): Promise<void> {
        await this.useFlow(HomeFlow).enter();
    }
}
```

## Module Public Contract

模块公开契约源文件：

```ts
// assets/modules/Home/code/public.ts
export interface HomeEnterParams {
    from?: string;
}
```

生成器写入首包 contract：

```ts
// assets/app/contracts/modules/Home.contract.generated.ts
export interface HomeEnterParams {
    from?: string;
}
```

业务代码 import contract，而不是 import 目标模块的 `code/public.ts`：

```ts
import type { HomeEnterParams } from 'yzforge-contracts/modules/Home';
```

## ModuleRef

首包轻量引用由生成器写入 `assets/app/registry/modules/Home.ref.generated.ts`，并通过 `yzforge/modules/Home` 暴露：

```ts
import type { HomeEnterParams } from 'yzforge-contracts/modules/Home';
import { defineModuleRef } from 'yzforge';

export const HomeRef = defineModuleRef<HomeEnterParams>({
    name: 'Home',
    bundle: 'yzforge-module-home',
    libraries: [],
});
```

如果模块依赖战斗底座，Ref 里声明轻量 Library 依赖：

```ts
import type { BattleEnterParams } from 'yzforge-contracts/modules/Battle';
import { BattleCoreRef } from 'yzforge/libraries/BattleCore';

export const BattleRef = defineModuleRef<BattleEnterParams>({
    name: 'Battle',
    bundle: 'yzforge-module-battle',
    libraries: [BattleCoreRef],
});
```

真实模块入口在模块 Bundle 内注册：

```ts
// assets/modules/Home/code/entry.generated.ts
// generated by YZForge
import { defineModuleEntry, registerModuleEntry } from 'yzforge';
import { HomeModule } from './HomeModule';
import { assets } from './assets.generated';
import { config } from './config.generated';

registerModuleEntry(defineModuleEntry({
    name: 'Home',
    type: HomeModule,
    assets,
    config,
    bundle: 'yzforge-module-home',
    libraries: [],
}));
```

## View

```ts
import { view } from 'yzforge';
import { Button } from 'cc';
import { PageHomeRefs } from './refs/PageHome.refs.generated';
import { HomeService } from '../service/HomeService';

@view()
export class PageHome extends PageHomeRefs {
    protected onOpen(): void {
        this.title.string = 'Home';
        this.listen(this.startButton.node, Button.EventType.CLICK, this.onClickStart, this);
    }

    private onClickStart(): void {
        this.module.useService(HomeService).startGame();
    }
}
```

## Service

```ts
import { Service } from 'yzforge';
import { PlayerModel } from '../model/PlayerModel';
import { PlayerLevelChanged } from '../events';

export class HomeService extends Service {
    public startGame(): void {
        const model = this.module.useModel(PlayerModel);
        model.started = true;

        this.module.event.emit(PlayerLevelChanged, {
            level: model.level,
        });
    }
}
```

Service 不直接打开 UI。需要打开 UI 时交给 Flow。

## Flow

```ts
import { Flow, isUiCancelResult } from 'yzforge';
import { assets } from '../assets.generated';

export class HomeFlow extends Flow {
    public async enter(): Promise<void> {
        await this.module.ui.open(assets.views.pageHome);
    }

    public async confirmExit(): Promise<void> {
        const result = await this.module.ui.openForResult(assets.views.popupConfirm, {
            title: 'Exit?',
        });

        if (isUiCancelResult(result)) {
            return;
        }

        await this.module.app.navigator.back();
    }
}
```

View 里的事件监听通过 `this.listen` 登记，View 关闭或销毁时由框架统一解绑。

## 资源实例化

普通运行时 prefab 由当前 Module 的 AssetScope 持有：

```ts
import { assets } from '../assets.generated';

const effect = await this.module.assets.instantiate(assets.runtime.effectHit, {
    parent: this.effectRoot,
});
```

Module 卸载时，框架会销毁已登记实例节点并释放对应资源。UI prefab 不走这个 API，必须使用 `module.ui.open`。

## Library Public Contract

Library 公开契约源文件：

```ts
// assets/libraries/BattleCore/code/public.ts
export interface DamageInput {
    attackerId: string;
    targetId: string;
}

export interface DamageSystemApi {
    calc(input: DamageInput): number;
}

export interface BattleCoreTokenMap {
    damageSystem: DamageSystemApi;
}
```

生成器写入首包 contract：

```ts
// assets/app/contracts/libraries/BattleCore.contract.generated.ts
import { defineLibraryTokens } from 'yzforge';

export interface DamageInput {
    attackerId: string;
    targetId: string;
}

export interface DamageSystemApi {
    calc(input: DamageInput): number;
}

export interface BattleCoreTokenMap {
    damageSystem: DamageSystemApi;
}

export const BattleCoreTokens = defineLibraryTokens<BattleCoreTokenMap>('BattleCore', {
    damageSystem: 'damageSystem',
});
```

## LibraryRef 与 Entry

首包轻量引用由生成器写入 `assets/app/registry/libraries/BattleCore.ref.generated.ts`，并通过 `yzforge/libraries/BattleCore` 暴露：

```ts
import { defineLibraryRef } from 'yzforge';

export const BattleCoreRef = defineLibraryRef({
    name: 'BattleCore',
    bundle: 'yzforge-lib-battle-core',
    libraries: [],
});
```

Library Bundle 内部真实入口：

```ts
// assets/libraries/BattleCore/code/entry.generated.ts
// generated by YZForge
import { classToken, defineLibraryEntry, registerLibraryEntry } from 'yzforge';
import { assets } from './assets.generated';
import { config } from './config.generated';
import { DamageSystem } from './system/DamageSystem';

registerLibraryEntry(defineLibraryEntry({
    name: 'BattleCore',
    refs: assets,
    config,
    bundle: 'yzforge-lib-battle-core',
    libraries: [],
    tokens: {
        damageSystem: classToken(DamageSystem),
    },
}));
```

模块使用 Library：

```ts
import { BattleCoreRef } from 'yzforge/libraries/BattleCore';
import { BattleCoreTokens } from 'yzforge-contracts/libraries/BattleCore';

const battleCore = await this.libraries.load(BattleCoreRef);
const damage = battleCore.use(BattleCoreTokens.damageSystem);
const value = damage.calc({
    attackerId: 'a',
    targetId: 'b',
});
```

模块不 import `DamageSystem` 实现类。

同一个 Module 内多次 `this.libraries.load(BattleCoreRef)` 返回同一个 `LoadedLibrary`，不重复增加 Library refCount。

## ContentPack

owner Module 内生成类型安全入口：

```ts
// assets/modules/Battle/code/content-packs.generated.ts
import { Prefab } from 'cc';
import { defineContentPack, contentPackAssetRef, contentPackConfigRef } from 'yzforge';
import { BattleCoreRef } from 'yzforge/libraries/BattleCore';

export const BattleLevel001ContentPack = defineContentPack({
    id: 'battle.level001',
    owner: 'Battle',
    bundle: 'yzforge-content-pack-battle-level001',
    libraries: [BattleCoreRef],
    refs: {
        levelRoot: contentPackAssetRef(Prefab, 'prefab/LevelRoot'),
        enemyTable: contentPackConfigRef('enemy'),
    },
});
```

模块加载关卡内容：

```ts
import { BattleLevel001ContentPack } from './content-packs.generated';
import { BattleFlow } from './flow/BattleFlow';

const contentPack = await this.contentPacks.load(BattleLevel001ContentPack);
const levelRoot = await contentPack.assets.instantiate(contentPack.refs.levelRoot, {
    parent: this.worldRoot,
});
await this.useFlow(BattleFlow).startLevel(contentPack, levelRoot);
```

ContentPack 不提供业务 API，只提供资源和配置。

## 跨模块进入

```ts
import { ActivitySpringRef } from 'yzforge/modules/ActivitySpring';

await app.preloadModule(ActivitySpringRef, {
    assets: true,
});

await app.enterModule(ActivitySpringRef, {
    activityId: 1001,
});
```

跨模块只允许通过 `ModuleRef` 做三件事：

- 预加载。
- 进入或打开。
- 卸载。

如果 A 模块需要 B 模块提供更细的业务能力，优先把这部分能力抽到 `Library`，而不是让 A 直接 import B 的 Service。

## Module Navigator

替换进入：

```ts
import { EnterMode } from 'yzforge';

await app.enterModule(HomeRef, undefined, {
    mode: EnterMode.Replace,
});
```

压栈进入：

```ts
import { EnterMode } from 'yzforge';

await app.enterModule(ActivitySpringRef, params, {
    mode: EnterMode.Push,
});

await app.navigator.back();
```

第一版同一时间只有一个前台 Active Module。Global UI 不进入模块栈。
